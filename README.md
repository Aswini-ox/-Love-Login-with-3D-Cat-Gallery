# 💕 Love Login with 3D Cat Gallery

A beautiful **love-themed login page** with floating hearts 💖.  
After a successful login, users are welcomed with a **3D rotating cat gallery** 🐱✨.

---

## 🚀 Features
- 🌈 Gradient background with smooth colors
- 💕 Floating animated hearts
- 🔐 Simple login system (username + password)
- 🖼️ 3D rotating cat photo gallery
- ✨ Attractive animations and transitions

---

## 🛠️ Tech Stack
- **HTML5** – Structure  
- **CSS3** – Styling, Animations, 3D Effects  
- **JavaScript** – Login logic & interactivity
